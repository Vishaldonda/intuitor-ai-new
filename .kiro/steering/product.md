# Product Overview

**SkillForge** - An interactive gamified learning platform for mastering Data Structures & Algorithms and web development (HTML, CSS, JavaScript).

## Core Features

- XP-based progression system with levels and streaks
- Multiple learning paths (DSA, HTML, CSS, JavaScript)
- Interactive question/challenge screens
- Achievement and progress tracking
- User profile with statistics
- Screen-based navigation (landing, dashboard, topics, questions, results, profile)

## User Flow

Landing → Dashboard → Topic Selection → Question/Challenge → Results → Profile
